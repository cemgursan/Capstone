Seaborn Joint Plot for Two Stock Prices

  Joint Plot from Seaborn consists of two different plots. First is the Scatter plot.
  Scatter plot shows us how two stock prices were at a specific point in time. In a scatter plot like this, it is possible to investigate how these two different stocks are related.
  If there is a significant correlation between two stocks, it is possible to make a prediction for one price from evaluating the other.

  Second plot in the Seaborn's Joint Plot is the Histogram plot which can be seen across the stockname.
  Histogram plot presents the different frequencies for different price ranges for these two stocks.

  In Jointplots, not only can you see the relationships between the two variables, but also how they are distributed individually.

Seaborn Pair Plot for Three Stock Prices

  Pair Plot from Seaborn also consists of two different plots: Scatter plot and Histogram
  This time, Scatter plot was utilized on the return percentages for different stocks. Every different two stocks create a scatter plot.
  By analyzing the clustering of different return percentage data point, we can understand which stock price fare better than the other.

  Second plot in the Seaborn's Pair Plog is again a Histogram plot
  Histogram plot can be found on the pair plot when the stock name is the same. Histogram plot shows the frequency of different return percentages for one stock.

  When these data and graphs are present, we can use linear regression analysis to check whether these stocks fit to a line.
