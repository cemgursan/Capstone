About this Project

   * Flask : To empower web applications with python
   * API : To get free WIKI stock data from Quandl
   * Pandas & Numpy : To perform data operations on downloaded data from API
   * Matplotlib & Seaborn : To visualize outcomes of data operations
   * Bokeh : To create interactive data visualizations for the web page
   * SQLite Database : To sign-up and log-in to authenticated sections of the website
   * Password Hashing: To encrypt passed data from users in the database for security
   * HTML, CSS & Javascripts : To visualize powerful web pages
   * Jinja : To connect HTML with python

Version 1.0

   * Only three stocks, Google, Apple, & Coca Cola were used from Quandl API
   * Free WIKI data set has been used from Quandl. The contributors has stopped updating this data set at 27-03-2018.
   * Bokeh is used for line plots only.
   * Other plots were done through matplotlib and seaborn and thus was sent to html as a picture
   * Password are protected through simple hashing in the database.
